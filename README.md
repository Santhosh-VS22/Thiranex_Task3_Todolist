# 📝 Thiranex Task 3 - JavaScript To-Do List

## 📌 Project Overview

This project is a responsive To-Do List web application developed as **Task 3** of the **Thiranex Web Development Internship**.

It demonstrates JavaScript DOM manipulation, event handling, local data persistence using Local Storage, and complete CRUD operations.

---

## 🚀 Features

- ✅ Add New Task
- ✅ Edit Existing Task
- ✅ Delete Task
- ✅ Mark Task as Completed
- ✅ Filter Tasks (All / Active / Completed)
- ✅ Automatic Data Storage using Local Storage
- ✅ Dynamic DOM Manipulation
- ✅ Event Delegation
- ✅ Responsive User Interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
